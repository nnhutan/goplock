# GoPlock

This is a Go/Fiber/Gorm project to build a blog.

# Setup

## Docker

This project uses Postgres and Redis. To run them in Docker, run the following
command:

```bash
docker-compose up -d
```

## Start the server

```bash
go run main.go
```

or if you have [Air](https://github.com/cosmtrek/air)

```bash
air
```
